# Security Policy

## Supported Versions

VivahGo is currently in an early release phase. Security updates are provided
for the active pre-1.0 release line only.

| Version | Supported          |
| ------- | ------------------ |
| 0.0.x   | :white_check_mark: |
| < 0.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in VivahGo, please report it by filling out this form:

**[Report a Security Vulnerability](https://forms.gle/1dLhpXTeSVciThMr5)**

We take all security reports seriously and will:
- Acknowledge receipt of your report within 72 hours
- Investigate and validate the vulnerability
- Provide an update on the resolution timeline
- Credit you in the security advisory (if desired)
